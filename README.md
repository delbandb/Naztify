# Naztify

Naztify is a private real-time mood messenger for two people.

## Run locally

```bash
npm install
npm start
```

## Deploy on Render

- Build command: `npm install`
- Start command: `npm start`
- Add environment variables:
  - `ONESIGNAL_APP_ID`
  - `ONESIGNAL_API_KEY`
  - `APP_URL`

## iPhone web push

- The receiver must add the site to the Home Screen
- She must open Naztify from the Home Screen icon
- She must tap the notifications button once to subscribe
